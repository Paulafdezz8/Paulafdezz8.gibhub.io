Tipos de selectores que entran en examen

1.Clase

Esta etiqueta clase selecciona todos los elementos que poseen esta etiqueta. Por ejemplo le pongo a este párrafo la clase "examen" y le ponemos el color rojo colorea a este párrafo y lo que ponga con esa clase de rojo,colocando en el style .examen.

2.Identificador (id=””)

Esta etiqueta solo selecciona el elemento que posee el id. Por ejemplo le pongo el id("selectores") al h2 y solo se puede poner a ese h2 con este id para poner color verde poniendo en style #selectores.

3.Todos los elementos (*)

Selecciona todos los elementos. Es decir le pongo * al texto para que me ponga fondo amarillo y me lo pone a todo el texto

4.Varios elementos ( p, em, h1, h2)

Todas las etiquetas que seleccionas en una lista van del mismo color. Le ponemos color azul a el h1 y a los em, poniendolos entre comas.

5.Elementos contenidos en otros elementos (ol li)

Selecciona todos los li que hay dentro del ol. Por ejemplo si en un ol quiero poner todos los elementos de morado lo hago de esta manera:

  1. desayuno
  2. como
  3. ceno

6.Sobrevuleo (p:hover)

Consiste en ponerle un color o fondo a un párrafo o título el cual solo se ve cuando pones el raton encima de él. Por ejemplo lo que hice en este párrafo.
